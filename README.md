# 教学立方辅助插件
- 该插件完全面向学生


## 安装方式

1. `clone`到本地（如果是`.zip`压缩包那解压成文件夹）
2. 打开Chrome或Chromium系浏览器，Chrome输入`chrome://extensions`，Chromium Edge输入`edge://extensions`
3. 打开“开发人员模式”
4. 点击“加载解压缩的扩展包”，选中本地的仓库文件夹

## 使用方式

+ 对学生：打开课件所在页面，**直接点击插件图标**，可以显示“下载”图标。
